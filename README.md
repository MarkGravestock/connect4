## Software Craftmanship 2016 Connect 4 Mini Project

A Test-Driven Development (TDD) kata for learning software craftsmanship principles.

Original kata: http://codemanship.co.uk/parlezuml/blog/?postid=1367

## Build and Test

This project uses:
- **Java 21** with Gradle build system
- **JUnit 5** (Jupiter) for testing
- **Hamcrest** for assertions
- **GitHub Actions** for continuous integration

### Running Tests Locally

```bash
.\gradlew.bat test
```

### Build Status

All tests are automatically run on every push and pull request. Test results are reported in the GitHub Actions workflow.
